## 亮点
- 这是 `0.3.0` 正式发布前的第四个测试打包版本，重点验证 GitHub Actions 构建出来的 macOS 安装包在另一台 Apple Silicon 设备上也能正常启动，不再出现主进程缺少模块的启动报错。

## 修复
- 修复了 macOS 打包版 Electron 主进程遗漏 `macAppVisibility.mjs` 的问题，安装后的应用启动时不应再弹出 `ERR_MODULE_NOT_FOUND` 并中断启动。
- 将该 macOS 可见性 helper 纳入 Electron 的正式编译产物链路，避免源码目录中的单独文件被构建脚本漏掉，确保 GitHub Actions 与本地打包都产出一致的安装包内容。

## 说明
- 这是预发布测试标签，主要用于复测 `rc3` 之后暴露出的 macOS 安装包启动故障，尤其是“GitHub Release 下载的 DMG 在另一台 Mac 上安装后立刻报错退出”的场景。
- 本次 GitHub Release 预期包含 `FlowSelect_0.3.0-rc4_browser_extension.zip`，用于继续验证预发布桌面端与扩展资源的打包一致性。
- macOS 本次重点验证 arm64 设备：从 GitHub Release 下载 DMG、拖入 `Applications`、首次启动应用、通过菜单栏/托盘入口唤起主窗口与设置窗口，以上链路都应正常工作。
