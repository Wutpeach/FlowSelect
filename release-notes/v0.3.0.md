## 亮点
- FlowSelect 桌面端已切换到 Electron 运行时与发布链路，带来了新的原生应用外壳、由 Node 托管的下载运行时、随版本发布的浏览器扩展资产，以及在开发环境和打包产物之间更一致的启动表现。
- 这一版显著扩展了网页视频采集覆盖范围。我们为更多站点接入了 gallery-dl 路由，扩展了下载提供方支持，并增强了 Instagram、YouTube、知乎等页面内的下载控件体验。
- 紧凑桌面 UI 进行了较大规模的打磨，包括更顺滑的图标模式切换、更稳定的拖拽与悬停行为、更清晰的队列与状态反馈，以及紧凑窗口、主窗口、托盘和设置页之间更一致的交互体验。
- macOS 打包与启动流程也经过了重点优化，涵盖紧凑模式一致性修复、打包后启动恢复、自定义 DMG 布局，以及多项首启和运行时稳定性改进。

## 修复
- 提升了下载链路稳定性，包括更一致的资源命名、粘贴链接后的标题恢复、直连下载跟随系统代理、更干净的任务清理逻辑、活动文案本地化，以及 Pinterest 等站点更稳定的处理方式。
- 修复了多项浏览器扩展与桌面端联动问题，包括应用重启后的 WebSocket 重连、首次点击下载触发不稳定、内部视频选择事件更安全，以及用于排查页面注入下载控件问题的调试面板。
- 强化了转码反馈与取消行为，加入更清晰的进度与 ETA 展示，优化最高质量 FFmpeg 参数，减少终端噪音，并在取消转码后避免后续状态异常。
- 解决了多项跨平台桌面壳层问题，包括 Windows 开机自启恢复、打包后渲染器加载修复、拖入文件夹后的输出路径处理、设置窗口更快显示，以及更稳健的启动与运行时引导流程。
- 收紧了 macOS 特有行为中的多个边界情况，包括紧凑窗口切换、托盘定位、输出目录交互、启动冻结与启动门控逻辑，让打包后的应用行为更接近开发态表现。

## 说明
- `0.3.0` 是基于 `0.2.9` 的一次重大桌面平台与下载能力更新；无需手动迁移。
- 带标签的正式发布仍应包含浏览器扩展产物 `FlowSelect_0.3.0_browser_extension.zip`。
