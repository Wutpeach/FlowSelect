## Highlights
- Migrated the desktop app onto the Electron release pipeline with native shell integrations, runtime bootstrap improvements, and faster packaged startup behavior across daily flows.
- Expanded web video collection coverage with stronger gallery-dl routing plus injected download controls for more sites, including Instagram, YouTube, and Zhihu player surfaces.

## Fixes
- Stabilized the compact window experience with smoother icon-to-main transitions, more reliable drag and hover behavior, and better queue/status feedback while downloads are active.
- Improved macOS packaging and startup resilience so packaged builds recover more reliably, keep tray and compact-mode behavior aligned, and reduce first-launch/runtime edge cases.

## Notes
- `0.3.0` is a broader desktop runtime and download-coverage release on top of `0.2.9`; no manual migration steps are required.
- Tagged releases should continue to include the browser extension asset as `FlowSelect_0.3.0_browser_extension.zip`.
