## 亮点
- 这是 `0.3.0` 正式发布前的第二个测试打包版本，重点验证 macOS 托盘形态、浏览器扩展预发布版本安装兼容性，以及完整 GitHub Release 构建链路。

## 修复
- 修复了浏览器扩展在 `0.3.0-rc2` 这类预发布版本下的 `manifest.version` 兼容问题，Chrome 现在会使用合法的数字版本号安装扩展，并通过 `version_name` 保留完整预发布标识。
- 调整了 macOS 应用启动形态，FlowSelect 现在按托盘 / 菜单栏工具模式运行，正常使用时不再在程序坞中显示应用图标。

## 说明
- 这是预发布测试标签，主要用于验证 `rc1` 之后补入的两项关键修复是否会影响三端打包与安装体验。
- 预发布构建仍应包含浏览器扩展产物 `FlowSelect_0.3.0-rc2_browser_extension.zip`，且解压后的扩展目录应可直接被 Chrome / Edge 正常加载。
- macOS 本次需要重点确认两个行为：应用图标不再出现在程序坞中，以及托盘入口仍可正常唤起主窗口和设置窗口。
