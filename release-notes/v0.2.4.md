## Highlights
- Added Pinterest video download support and improved the bundled downloader workflow so more video cards can be collected reliably.
- Expanded Settings with bundled downloader information and a richer animated card layout for downloader status.

## Fixes
- Stabilized Pinterest card downloads across masonry cards, detail actions, drag-enriched cards, and desktop drag hint fallbacks.
- Improved runtime packaging by unifying the bundled binary layout, bundling FFmpeg for portable builds, and tightening sidecar maintenance behavior.
- Fixed extension reconnect handling so video selection can retry cleanly after the app reconnects.
- Reduced popup window instability by migrating the UI motion layer and hardening popup behavior.

## Notes
