## Highlights
- 这是 `0.3.0` 正式发布前的测试打包版本，用来验证当前 Electron 桌面构建、浏览器扩展资产和 GitHub Release 流程是否都能按预期产出。
- 本次测试版已经包含 `0.3.0` 基线能力，以及最近补发的下载完成后转码恢复、Bilibili 兼容预览避免重复转码、完成勾号稳定留在主窗口等修复。

## Fixes
- 恢复了 Electron 版下载完成后的后续转码链路，最高质量但仍需转码的视频现在会继续进入转码流程，而不是停在源文件下载完成。
- 优化了 Bilibili 预览视频在兼容格式下的处理方式，避免本来就可直接使用的素材再被多做一次不必要的转码。
- 修复了下载或转码完成后的紧凑窗口状态竞态，完成勾号现在会稳定显示在主窗口里，不会偶发落到图标模式下。

## Notes
- 这是预发布测试标签，仅用于验证 `0.3.0` 的重新打包流程，不代表最终正式版已经重新发布。
- 如果本次打包结果正常，后续再重新创建正式 `v0.3.0` 标签并发布正式产物。
- 预发布构建仍应包含浏览器扩展产物 `FlowSelect_0.3.0-rc1_browser_extension.zip`。
