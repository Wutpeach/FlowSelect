## 亮点
- 这是 `0.3.0` 正式发布前的第六个测试打包版本，重点验证桌面端几个高频交互修复：快捷键唤起位置恢复、紧凑模式下位置保持一致、主窗口拖拽体验，以及支持日志导出的稳定性。

## 修复
- 修复了通过快捷键唤起主窗口时的位置异常问题，窗口应当恢复到更符合预期的位置，不再频繁偏移。
- 修复了紧凑模式在快捷键唤起后容易跑位的问题，紧凑模式现在会尽量保持在原来的快捷入口位置附近。
- 修复了下载完成或拖放操作之后主窗口拖拽行为被打断的问题，面板在这些操作后仍应可以正常拖动。
- 优化了支持日志导出链路，导出过程对桌面端环境的兼容性和可用性更好，便于继续收集问题诊断信息。

## 说明
- 这是预发布测试标签，主要用于回归验证桌面端窗口定位与拖拽体验，尤其是快捷键唤起、紧凑模式位置保持，以及支持日志导出是否稳定可用。
- 本次 GitHub Release 预期包含 `FlowSelect_0.3.0-rc6_browser_extension.zip`，用于继续验证桌面端与扩展联动链路在预发布版本中的一致性。
