## Highlights
- Added Chinese-first localization across the desktop windows, browser extension popup, and native tray/menu surfaces.
- Improved portable Bilibili and other yt-dlp downloads so bundled downloads behave more consistently across different Windows machines.

## Fixes
- Prevented the bundled yt-dlp runtime from inheriting host-machine yt-dlp config files, which could make the same portable build behave differently on different PCs.
- Added automatic recovery for stale yt-dlp resume state that could surface `HTTP Error 416: Requested Range Not Satisfiable` during `highest` downloads.
- Hardened runtime language-change logging and localization integration so translated surfaces stay aligned more reliably.

## Notes
- This release focuses on shipping the new localization foundation together with a portable download reliability fix for packaged builds.
- The compare link uses `v0.2.2...v0.2.5` because the repository currently has no `v0.2.3` or `v0.2.4` tag.
