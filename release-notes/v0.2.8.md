## Highlights
- Added direct clipboard image paste support so copied screenshots and images can be sent into FlowSelect without saving an intermediate file first.
- Added an in-app update entry and smoother packaged runtime preparation so desktop installs can surface FlowSelect app updates and come up ready faster.
- Polished the X injected download button so its cat icon action better matches the native action-button feel on video posts.

## Fixes
- Stopped a noisy startup `yt-dlp` refresh loop that could make the main window feel unsettled right after launch.
- Tightened compact settings and status chrome so version and downloader-maintenance surfaces stay clearer in packaged builds.

## Notes
- `0.2.8` rolls the `0.2.7` prerelease updater and packaging work into the stable line.
- Windows packaged installs now prewarm managed runtimes during install, which should reduce first-launch setup friction on new machines.
