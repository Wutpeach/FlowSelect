## Highlights
- Refined the compact task queue so download and transcode work is easier to monitor, with clearer status copy and stronger separation between download and MP4 compatibility steps.
- Polished settings cards and shared desktop controls so the main window and Settings surface feel more consistent across buttons, fields, toggles, and queue actions.

- Improved the "exit current transcode" flow so cancelling an active compatibility conversion keeps queue state and task feedback aligned.
