## 亮点
- 这是 `0.3.0` 正式发布前的第三个测试打包版本，重点验证 macOS 仅发布 Apple Silicon 安装包后的发版链路是否稳定，以及最近两项安装体验修复是否都能正常进入测试包。

## 修复
- 修复了浏览器扩展在 `0.3.0-rc*` 这类预发布版本下的版本号兼容问题，Chrome / Edge 现在可以正常加载测试版扩展包，同时保留完整预发布标识用于识别。
- 调整了 macOS 应用启动形态，FlowSelect 现在按托盘 / 菜单栏工具模式运行，正常使用时不再在程序坞中显示应用图标。
- 调整了 GitHub Release 构建矩阵，macOS 现在只发布 arm64 安装包与 ZIP，不再产出 Intel 安装包，避免无效构建和额外失败点。

## 说明
- 这是预发布测试标签，主要用于验证新的 macOS arm64-only 发版流程，以及 `rc2` 暴露出的构建链路问题是否已经清理干净。
- 本次 GitHub Release 预期包含 `FlowSelect_0.3.0-rc3_browser_extension.zip`，解压后的扩展目录应可直接被 Chrome / Edge 加载。
- macOS 本次重点只验证 arm64 设备：应用图标不出现在程序坞中、托盘入口仍可正常唤起主窗口和设置窗口、DMG 内仅附带 arm64 相关桌面产物。
