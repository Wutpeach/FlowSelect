FlowSelect macOS install

1. Drag FlowSelect.app into Applications.
2. Open FlowSelect from Applications.
3. If macOS blocks the app on first launch:
   - Right-click FlowSelect.app and choose Open, or
   - Go to System Settings > Privacy & Security and allow the app, or
   - Open Terminal and run the manual repair command below after the app is already installed in Applications.

Advanced manual repair command:
xattr -dr com.apple.quarantine "/Applications/FlowSelect.app"

Notes:
- The manual repair command only repairs an installed copy of FlowSelect.app.
- It does not perform the drag-to-Applications install step for you.
- If you installed FlowSelect somewhere else, adjust the app path in the manual command.
